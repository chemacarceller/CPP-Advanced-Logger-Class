# Advanced-C++-Logger-Class

MyLogger is an advanced C++ class for creating a log file.

It is designed to be used from a Python script.

The class acts as a singleton, so it is useful in large programs (e.g., desktop applications) where this class must be instantiated from different parts so that only a single object is instantiated.

=================================================================

Clase MyLogger avanzada escrita en C++ para la creación de un registro de logs

Esta desarrollado para ser utilizado desde un script de Python

La clase actua como singleton, por lo que es util en programas grandes (por ejemplo aplicaciones de escritorio) donde se debe instanciar esta clase desde distintas partes de forma que únicamente se instancie un único objeto
